00x02 注意事项

  • 需要伪造一下header头和cookie，否则提示非法访问…
  • 获取的源码需要encoding编码成 utf-8 来防止中文乱码，当然有时候是其他编码，如GBK
  • 以二进制形式保存文件的话，f.write()需要使用res.content而不是res.text
